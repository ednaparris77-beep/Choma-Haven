 Choma Haven

Simple nyama choma restaurant website built with HTML, CSS & JavaScript.

Live demo: https://ednaparris77-beep.github.io/Choma-Haven/

Features
- Responsive navbar & menu grid
- Hero background
- Food photos (beef, goat, chicken, sides, drinks)

Run Locally
1. Clone: `git clone https://github.com/ednaparris77-beep/Choma-Haven.git`
2. Open `index.html` (or use VS Code Live Server)

Deploy (GitHub Pages)
Repo Settings → Pages → Source: main branch → / (root) → Save  
Site URL: https://ednaparris77-beep.github.io/Choma-Haven/
